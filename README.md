   ## The project
   The project consists to re-create the classic dino game which launched when connection lost in console.
   The datas are saved in local and the game can be download from the launcher.

   ## Features
   Save data in local   <br />
   Jump, crouch     <br />
   See scoreboard   <br />
   Customize textures
